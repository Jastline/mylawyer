## Как скачать?

1. Запуск flutter clean
2. Запуск flutter pub get
3. flutter build apk --no-shrink
4. cd "C:\Users\JastLIne\AndroidStudioProjects\mylawyer\build\app\outputs\flutter-apk"
5. adb install app-release.apk
